# 🎮 Hangman — Terminal Word Game

A fun terminal-based Hangman game built in Python with ASCII art, word categories, hints, and a play-again loop.

---

## ✨ Features

- 🎨 **ASCII hangman art** — watch the figure build with every wrong guess
- 📂 **4 word categories** — Tech, Animals, Science, Food
- 💡 **Hints** — every word comes with a clue so it's fair
- 🔁 **Play again** — no need to restart the script
- ✅ **Input validation** — handles duplicate guesses and invalid input cleanly

---

## 🚀 How to Run

Make sure you have Python installed (3.6 or above).

```bash
python hangman.py
```

---

## 🕹️ How to Play

1. Choose a category (Tech, Animals, Science, Food, or Random)
2. A hidden word is chosen — you see blanks like `_ _ _ _ _ _`
3. Guess one letter at a time
4. You have **6 lives** — every wrong guess draws part of the hangman
5. Reveal the full word before the hangman is complete to win!
